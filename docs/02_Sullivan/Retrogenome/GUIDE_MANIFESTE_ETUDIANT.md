# Manifeste HCI — Guide Étudiant (AetherFlow Retro-Genome)

> **Note** : Le manifeste est facultatif. Sans lui, le système inférera l'intention depuis le visuel seul. Avec lui, le diagnostic sera plus précis.

---

## Structure du Manifeste

### 1. Ce que fait l'interface (en une phrase)
"Cette interface permet à [qui] de [faire quoi] dans [quel contexte]."
*Exemple classique : "Cette interface permet à un livreur de gérer ses missions en temps réel."*
*Exemple expérimental : "Cette interface permet à un lecteur d'explorer une histoire selon les lois du carré magique de Lo Shu."*

### 2. Organisation des zones
Pour chaque zone de l'écran, décrivez son rôle :
- **Zone A** : [Nom] → Ce qu'elle affiche / permet de faire
- **Zone B** : ...
*(Nommez-les comme vous voulez : "panneau gauche", "bloc central", "colonne 3", "secteur nord"...)*

### 3. Principe de navigation / flux
Comment l'utilisateur passe d'un état à un autre ?
- "En cliquant sur [X], on accède à [Y]."
- "La liste à gauche filtre le contenu central."
- "Les cellules sont interconnectées selon [règle spécifique]."

### 4. Éléments intentionnellement absents
Ce que vous avez choisi de ne PAS mettre (et pourquoi), ou ce qui n'est pas encore implémenté :
- "Pas de barre de navigation globale — le système est délibérément sans retour arrière."
- "Le formulaire de connexion est prévu mais absent de cette version."

### 5. Contrainte ou parti-pris (facultatif)
Un principe fort qui gouverne l'ensemble :
- **Contrainte formelle** (ex. "Toutes les vues suivent une grille de 3×3")
- **Référence théorique** (ex. "Basé sur les principes de Jakob Nielsen")
- **Paradigme expérimental** (ex. "Le principe péréquien s'applique à la navigation")

---

## Pourquoi remplir ce manifeste ?

Le système **Retro-Genome** met en tension votre réalisation (PNG) et vos intentions (Manifeste). 
- **Matches** : Ce qui est compris et validé.
- **Gaps** : Ce que vous aviez prévu mais qui n'est pas visible (ou mal exprimé).
- **Surplus** : Ce qui est présent visuellement mais n'a pas de justification déclarée.
