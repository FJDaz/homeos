# Design System HoméOS

## 1. Vision & Principes
HoméOS est un système de design moderne, fluide et technique, conçu pour offrir une clarté absolue dans les interfaces d'édition et d'audit. Il repose sur trois piliers :
- **Clarté Technique** : Une typographie sans-empattement robuste pour une lecture sans effort.
- **Douceur Géométrique** : L'utilisation de rayons de courbure généreux pour humaniser l'aspect technique.
- **Accents de Précision** : Un vert vif utilisé stratégiquement pour guider l'action et l'attention.

## 2. Fondations Visuelles

### Palette de Couleurs
- **Primary (Vert HomeOS)** : `#A3CD54` - Utilisé pour les boutons d'action principale, les états actifs et les indicateurs de succès.
- **Backgrounds** : Blanc pur et gris ultra-léger pour la structure.
- **Texts** : Gris anthracite profond pour le contenu principal, gris moyen pour les informations secondaires.

### Typographie
- **Police de caractères** : `Source Sans 3`.
- **Hiérarchie** :
    - **Display/Headlines** : Semi-bold, espacement standard.
    - **Body** : Regular, interlignage aéré (1.5).
    - **Interface/Labels** : All-caps ou Bold pour les éléments de navigation.

### Formes & Structure (Shape)
- **Border Radius** : `20px` systématique sur les conteneurs principaux, les modales et les boutons.
- **Bordures** : `1px` solide, couleurs subtiles pour la séparation des zones sans alourdir.
- **Grille de points** : Utilisée en arrière-plan du canevas pour renforcer l'aspect "outil d'édition".

## 3. Composants Clés

### Boutons
- **Action (HomeOS)** : Fond vert `#A3CD54`, texte blanc ou sombre, arrondi de 20px.
- **Ghost/Tertiary** : Bordure subtile, texte gris, sans fond.

### Conteneurs (Cards)
- Fond blanc, bordure de 1px, arrondi de 20px. Ombrages très légers (soft shadows) pour la profondeur.

### Chat Interface (Sullivan)
- Bulles de chat avec arrondis de 20px, typographie claire, champ de saisie avec icônes d'action intégrées.
