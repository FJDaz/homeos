# 🧠 Rapport d'Intelligence : UI/UX PRO MAX

Ce rapport documente les capacités et la structure du skill `ui-ux-pro-max` intégré à l'écosystème **AetherFlow**. Ce module sert de "Directeur Artistique" numérique pour garantir que toute génération d'UI (Generative UI) respecte les standards les plus élevés du marché.

## 🏮 Nature du Module
`ui-ux-pro-max` n'est pas une simple documentation statique; c'est un moteur de recherche relationnel basé sur **12 bases de données (CSV)** totalisant des centaines d'entrées d'expertise design.

### 🗃️ Répertoire des Données (Intelligence Atomique)
| Domaine | Contenu & Expertise |
| :--- | :--- |
| **`products.csv`** | Matrice de décision par type de produit (SaaS, FinTech, E-commerce). |
| **`styles.csv`** | 67 styles complets (Minimalism, Glassmorphism, Brutalism, etc.). |
| **`ui-reasoning.csv`** | Règles logiques pour choisir un design selon le secteur d'activité. |
| **`ux-guidelines.csv`** | 99 principes de navigation, accessibilité et ergonomie. |
| **`typography.csv`** | 57 pairings de Google Fonts optimisés pour la lisibilité. |
| **`colors.csv`** | 96 palettes chromatiques prêtes à l'emploi. |
| **`react-performance.csv`** | Guide d'optimisation spécifique pour les interfaces réactives. |

## 🚀 Fonctionnalités Clés pour AetherFlow

### 1. Génération de Design System (Architectural)
Le module permet de générer instantanément un **Master Design System** comprenant :
- **Mood & Vision** : L'intention émotionnelle de l'interface.
- **Tokens** : Couleurs, typographies et effets de bord.
- **Anti-patterns** : Une liste de "Ce qu'il ne faut PAS faire" (ex: pas d'emojis, pas de transitions brusques).

### 2. Protocole de Persistance (Hierarchical Retrieval)
Le skill supporte un mode de persistance qui sépare :
- `MASTER.md` : Les règles globales du projet AetherFlow.
- `Overriders` : Des règles spécifiques par page ou par organe.

### 3. Checklist de Validation (Directeur Artistique)
Avant chaque livraison, le module impose une vérification de la "Qualité Visuelle" :
- [ ] Icônes SVG pures uniquement (Lucide / Heroicons).
- [ ] Hover states stables et contrastés.
- [ ] Transitions fluides (150-300ms).
- [ ] Accessibilité (Contraste 4.5:1 minimum).

## 🛡️ Gestion de la Redondance (Les "Doublons")
Le module présente des redondances volontaires dans sa documentation technique pour :
1. **Fiabilité de l'IA** : Assurer que l'agent capture l'information critique quel que soit son mode de lecture.
2. **Prêt-à-l'emploi** : Fournir des snippets de code (ex: Tailwind) directement dans les sections concernées sans forcer l'agent à naviguer entre 10 fichiers.
