## 2. Les Organes : Les Fonctions Vitales

Les organes sont les blocs logiques complexes. Ils ne sont pas juste des "boutons", mais des systèmes d'interaction entre l'utilisateur et les agents IA.

* **Le Moniteur d'Orchestration** : Un panneau qui affiche l'exécution d'un plan en temps réel. Il permet de voir si l'agent est en mode `FAST`, `BUILD` ou `DOUBLE-CHECK`.
* **Le Dialogueur Sullivan (Chatbot)** : Un organe de conversation pour affiner tes composants frontend par la voix ou le texte.
* **Le Réviseur de Code (Surgical Editor)** : Un organe qui montre les modifications ciblées (le "patch") au lieu de remplacer tout un fichier.
* **La Balance des Coûts** : Un petit widget discret qui affiche les jetons (tokens) consommés et le coût estimé pour garder le contrôle sur ton budget.

* **Entrées d'Intention** : Zones de texte intelligentes qui comprennent tes besoins sans jargon technique.
* **Indicateurs d'Homéostasie** : Des badges de couleur (Vert/Orange/Rouge) pour la santé du service (Health check).
* **Sélecteurs de Workflow** : Des commutateurs pour choisir entre la vitesse (`-q` Proto) ou la qualité absolue (`-f` Prod).
* **Cartes de Prévisualisation** : Pour voir tes composants Sullivan avant de les intégrer.
